## Быстрый ориентир по проекту

**Суть проекта:** Telegram-бот для VPN-сервиса. Бот показывает экраны с кнопками, управляет подписками и тарифами.

**Ключевая идея архитектуры:** экраны (меню, FAQ, тарифы) хранятся в базе данных. Каждый экран может иметь привязанный хендлер для динамической логики. Навигация — через кнопки, которые ссылаются на другие экраны по ключу.

**Точка входа:** `POST /api/telegram/webhook` → `WebhookController` → `BotService`

**5 моделей:** `User`, `Screen`, `ScreenButton`, `UserState`, `Payment`

**3 хендлера:** `MainMenuHandler`, `ProfileHandler`, `TariffHandler`

**4 сервиса:** `BotService`, `UserService`, `TariffService`, `ConfigService`
